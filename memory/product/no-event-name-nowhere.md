Never use the name **"Nowhere"** (or "Nowhere Festival") in UI text, emails, docs, release notes, or any user-facing content. **"Elsewhere" is the current event name and is 100% fine to use.**

**Why:** Legal reasons specific to the "Nowhere" name. (Earlier guidance had over-broadened this to all event names — that was wrong; only "Nowhere" is restricted.)

**How to apply:** When writing or reviewing user-facing text, never use "Nowhere". "Elsewhere" can be used freely. If in doubt about another name, ask before using it.
