How do I sign up for shifts? I can see the Shifts page but nothing seems clickable.

## Answer

## Signing up for shifts

The most likely reason nothing is clickable is that **shift browsing is currently closed** for regular volunteers. When an Admin turns off the browsing toggle, you can only see shifts if you already have existing signups — everything else appears but isn't interactive.

**What you can do:**

1. **Check back later** — browsing opens and closes based on event scheduling decisions made by Admin.
2. **Contact your Build team coordinator** — coordinators can always browse and can voluntell you onto a shift directly, which would then give you a signup and unlock the view for you.

Once browsing is open, you'll be able to click a shift slot and sign up directly from the Shifts page.

If browsing *is* open and things are still unclickable, it's worth submitting feedback via the feedback widget so a coordinator can investigate.
